Direct Stat v1.0 (11.08.2019)

Первая публичная версия скрипта.
Позволяет получить основные данные по рекламным кампаниям из Яндекс Директ

Для работы распакуйте архив direct stat v1.0 и запустите файл direct-stat public.exe
Программа всегда запрашивает токен для доступа по API. Вам необходимо получить его. Информация в справке яндекса https://yandex.ru/dev/direct/doc/start/token-docpage/

Построенные отчеты сохраняются в папку reports в директории скрипта.
Также сохраняются сырые данные в формате .tsv в папке raw-data

Эти папки удалять не следует, иначе скрипт будет выдавать ошибку.
Это моя первая работа, в дальнейшем планирую улучшить скрипт (сохранение используемого токена, построение отчета в единый файл)
